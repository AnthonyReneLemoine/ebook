Configuration Google

Créez une application Google OAuth et renseignez votre Client ID. Cette opération est à faire une seule fois.

Rendez-vous sur Google Cloud Console et créez un projet (ex : liseuse).

Dans APIs & Services → Bibliothèque, activez l'API Google Drive.

Dans Identifiants, créez un ID client OAuth 2.0 de type Application Web. Ajoutez l'URL GitHub Pages dans Origines JavaScript autorisées.

Dans Écran de consentement OAuth, ajoutez votre Gmail en Utilisateurs test.

Copiez l'ID client (se termine par .apps.googleusercontent.com) ci-dessous.

Stocké uniquement dans votre navigateur (localStorage).
