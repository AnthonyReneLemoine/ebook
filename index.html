Configuration Google

Créez une application OAuth pour connecter votre Google Drive. À faire une seule fois.

1

Rendez-vous sur Google Cloud Console → créez un projet (ex : liseuse).

2

APIs & Services → Bibliothèque → activez l'API Google Drive.

3

Identifiants → Créer un ID client OAuth 2.0 (Application Web). Ajoutez l'URL GitHub Pages dans Origines autorisées.

4

Écran de consentement → ajoutez votre Gmail en Utilisateurs test.

5

Copiez l'ID client (se termine par .apps.googleusercontent.com).

Stocké localement dans votre navigateur uniquement.
