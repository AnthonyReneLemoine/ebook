Configuration Google

Pour utiliser la liseuse, vous devez créer une application Google OAuth et renseigner votre Client ID. Cette opération est à faire une seule fois.

Rendez-vous sur Google Cloud Console et créez un nouveau projet (ex : liseuse).

Dans APIs & Services → Bibliothèque, activez l'API Google Drive.

Dans APIs & Services → Identifiants, créez un ID client OAuth 2.0 de type Application Web. Ajoutez l'URL de votre GitHub Pages dans Origines JavaScript autorisées (ex : https://VotreLogin.github.io).

Dans Écran de consentement OAuth, ajoutez votre adresse Gmail en Utilisateurs test.

Copiez l'ID client (se termine par .apps.googleusercontent.com) et collez-le ci-dessous.

Stocké uniquement dans votre navigateur (localStorage). Non transmis à des tiers.
Un dossier sera créé à la racine de votre Drive avec ce nom.
